12、Aggregates

  • COUNT returns number of values in specified column.
  • SUM returns sum of values in specified column.
  • AVG returns average of values in specified column.
  • MIN returns smallest value in specified column.
  • MAX returns largest value in specified column.

（1）Use of COUNT(*)

  • How many properties cost more than £350 per month to rent?

（2）Use of COUNT(DISTINCT)

  • How many different properties viewed in May 2004?

（4）Use of MIN, MAX, AVG

  • Find minimum, maximum, and average staff salary.

13、Grouping

[!NOTE]

All column names in SELECT list must appear in GROUP BY clause unless name is used only in an aggregate function. （SELECT 子句中的所有列除非用在聚集函数中，否则必须在 GROUP BY 子句中出现。）

必须先使用where字句，再使用group by字句

HAVING clause is designed for use with GROUP BY to restrict groups that appear in final result table.（HAVING 子句的设计意图是与 GROUP BY 子句一起使用，来限定哪些分组将出现在最终查询结果中。）

Column names in HAVING clause must also appear in the GROUP BY list or be contained within an aggregate function.（HAVING 子句使用的列名必须出现在 GROUP BY 子句列表中，或包括在聚集函数中。）

（1）Use of GROUP BY

  • Find number of staff in each branch and their total salaries.

（2）Use of HAVING

  • For each branch with more than 1 member of staff, find number of staff in each branch and sum of their salaries.

14、Subqueries（嵌套查询）

[!IMPORTANT]

ORDER BY clause may not be used in a subquery (although it may be used in outermost SELECT).（ORDER BY 子句不能用于子查询（虽然可用在最外面的 SELECT 语句中））。

Subquery SELECT list must consist of a single column name or expression, except for subqueries that use EXISTS.（子查询 SELECT 列表必须由单个列名或表达式组成，除非子查询使用了关键词EXISTS）

When subquery is an operand in a comparison, subquery must appear on right-hand side.（当子查询是比较表达式中的一个操作数时，子查询必须出现在表达式的右面。）

（1）Subquery with Equality

  • List staff who work in branch at ‘163 Main St’.
  • List all staff whose salary is greater than the average salary, and show by how much.

（2）Nested subquery: use of IN

  • List properties handled by staff at ‘163 Main St’

15、ANY and ALL

[!IMPORTANT]

ANY and ALL may be used with subqueries that produce a single column of numbers.（关键字 ANY 和 ALL 用于产生单个列的子查询。）

With ALL, condition will only be true if it is satisfied by all values produced by subquery. （若子查询前缀关键字 ALL，那么仅当子查询产生的所有值都满足条件时，条件才为真。）

With ANY, condition will be true if it is satisfied by any values produced by subquery. （若子查询前缀关键字 ANY，那么子查询产生的任何一个值（一个或多个）满足条件时，条件就为真。）

If subquery is empty, ALL returns true, ANY returns false.

（1）Use of ANY/SOME

  • Find staff whose salary is larger than salary of at least one member of staff at branch B003.

（2）Use of ALL

  • Find staff whose salary is larger than salary of every member of staff at branch B003.
